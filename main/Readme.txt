
This code executes prediction models for the number of cases of COVD-19

execution

execute states_csv to generate models from brazil and its states
execute municipios_csv to generate models of Brazilian municipalities

execution parameters
min_cases => minimum number of cases to run the model
min_dias => minimum number of days since the first case to run the model
arq_saida => prediction output file
arq_sumario_saida => output file with the models summary
previsao_ate => date for which predictions are made

input file

datamun.csv => file with information about the municipalities
datastate.csv => file with information about the states
